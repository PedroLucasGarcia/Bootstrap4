Em Bootstrap, um "container" é uma classe CSS que ajuda a estruturar o layout da página. Existem duas principais classes de container no Bootstrap: .container e .container-fluid. Ambas são usadas para envolver o conteúdo da página e aplicar estilos específicos para tornar o layout responsivo.


.container

É um container fixo que tem uma largura máxima e se ajusta automaticamente às diferentes larguras de tela. O conteúdo dentro de um .container é centralizado horizontalmente. Este é frequentemente utilizado para layouts mais tradicionais.


.container-fluid

É um container que ocupa 100% da largura da tela, independentemente do tamanho do dispositivo. Isso é útil para criar layouts que se expandem para preencher toda a largura da tela.
