HTML

A linguagem para construir páginas web

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar e estruturar conteúdo na web. Ele é a espinha dorsal da maioria das páginas da internet, sendo a base para a construção de documentos hipermídia. HTML permite que desenvolvedores descrevam e organizem o conteúdo de uma página, indicando a função de cada elemento presente.

CSS

A linguagem para estilizar páginas web

CSS é responsável pela apresentação visual das páginas web. Ao criar regras de estilo para os elementos HTML, como cores, fontes e layouts, CSS proporciona um controle preciso sobre o design das páginas. Além disso, permite a criação de páginas responsivas, adaptando-se a diferentes dispositivos e tamanhos de tela.

JavaScript

A linguagem para programar páginas web

JavaScript é uma linguagem de programação do lado do cliente, permitindo interatividade dinâmica nas páginas web. Utilizado para manipular elementos HTML, responder a eventos do usuário e realizar operações assíncronas, o JavaScript contribui para a criação de experiências interativas e dinâmicas na web.

Aprenda Javascript
PHP

Uma linguagem de programação de servidor web

PHP é uma linguagem de script do lado do servidor, amplamente utilizada para desenvolver aplicativos web dinâmicos. Responsável por processar dados no servidor antes de enviá-los ao navegador, o PHP é utilizado para interações com bancos de dados, gerenciamento de sessões e criação de páginas dinâmicas.

Aprenda PHP
Bootstrap

Um framework CSS para projetar melhores páginas da web

Bootstrap é um framework front-end que simplifica o desenvolvimento web. Oferece um conjunto de estilos predefinidos, componentes e grid system, permitindo a criação rápida e consistente de interfaces de usuário responsivas. Ao facilitar o design e a implementação, o Bootstrap acelera o processo de desenvolvimento web.
